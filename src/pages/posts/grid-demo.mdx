# Grid Component Demo

## Two Column Grid (Default)

<Grid>
  <Card type="success" title="First Item">
    This is the first item in a two-column grid layout.
  </Card>

<Card type="info" title="Second Item">
  This is the second item in the grid.
</Card>

<Card type="warning" title="Third Item">
  This is the third item. The grid wraps to a new row.
</Card>

  <Card type="error" title="Fourth Item">
    This is the fourth item, completing the second row.
  </Card>
</Grid>

## Three Column Grid

<Grid cols={3}>
  <Card title="Column 1">First column content</Card>

<Card title="Column 2">Second column content</Card>

  <Card title="Column 3">Third column content</Card>
</Grid>

## Single Column Grid

<Grid cols={1}>
  <Card type="info" title="Full Width Card">
    This card takes up the full width of the container.
  </Card>

  <Card type="success" title="Another Full Width Card">
    Grid components are responsive and stack on mobile.
  </Card>
</Grid>

## Custom Span Grid (2/3 + 1/3)

<Grid cols={3}>
  <GridItem cols={2}>
    <Card type="info" title="Wide Card (2/3)">
      This card spans 2 columns in a 3-column grid, taking up 2/3 of the width.
    </Card>
  </GridItem>

  <GridItem cols={1}>
    <Card type="warning" title="Narrow (1/3)">
      This card spans 1 column, taking 1/3 of the width.
    </Card>
  </GridItem>
</Grid>

## Another Custom Layout (1/3 + 2/3)

<Grid cols={3}>
  <GridItem cols={1}>
    <Card title="Sidebar">
      Narrow sidebar content (1/3)
    </Card>
  </GridItem>

  <GridItem cols={2}>
    <Card type="success" title="Main Content">
      Main content area taking up 2/3 of the space. You can put longer content here.
    </Card>
  </GridItem>
</Grid>
