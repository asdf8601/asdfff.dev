# Mermaid Diagrams Demo

## Flowchart

<Mermaid>
```
flowchart TD
    A[Start] --> B{Is it working?}
    B -->|Yes| C[Great!]
    B -->|No| D[Debug]
    D --> E[Fix bugs]
    E --> B
    C --> F[Deploy]
    F --> G[End]
```
</Mermaid>

## Sequence Diagram

<Mermaid>
  {`
sequenceDiagram
    participant User
    participant Browser
    participant Server
    participant Database
    
    User->>Browser: Enter URL
    Browser->>Server: HTTP Request
    Server->>Database: Query data
    Database-->>Server: Return results
    Server-->>Browser: HTTP Response
    Browser-->>User: Display page
`}
</Mermaid>

## Class Diagram

<Mermaid>
  {`
classDiagram
    class Animal {
        +String name
        +int age
        +makeSound()
    }
    class Dog {
        +String breed
        +bark()
    }
    class Cat {
        +String color
        +meow()
    }
    Animal <|-- Dog
    Animal <|-- Cat
`}
</Mermaid>

## State Diagram

    Processing --> Success : Complete
    Processing --> Error : Fail
    Success --> [*]

    Retry --> Processing
    Error --> [*] : Give up
`}
</Mermaid>
